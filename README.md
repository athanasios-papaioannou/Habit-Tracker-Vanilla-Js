# 📅 Habit Tracker

A Vanilla JavaScript habit tracker focused on daily consistency, streak tracking and localStorage persistence. Built as a case study to strengthen core frontend development concepts.

---

## ✨ Features

- Add and delete habits
- Daily streak tracking
- Progress indicator
- LocalStorage persistence
- Dark / Light mode
- Clean user interface

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- DOM API
- Browser Local Storage

---

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/athanasios-papaioannou/Habit-Tracker-Vanilla-Js.git
```

Open the project

```bash
cd Habit-Tracker-Vanilla-Js
```

Launch

Simply open `index.html` in your browser.

## 🧠 Key Learnings

During this project I focused on understanding the reasoning behind every implementation rather than simply making the application work.

Some of the concepts explored include:

- Application State Management
- Single Source of Truth (SSOT)
- DOM Manipulation
- Event Handling
- Local Storage Persistence
- Dynamic Rendering
- Data Modelling
- UX Improvements
